# StickyNote

Windows 上的极简便签应用，基于 WPF/.NET 8 构建。采用**单窗口多标签**设计，支持自动保存、智能待办排序、系统托盘集成与全局快捷键，专注于提供流畅、高效的记录体验。

## ✨ 主要特性

*   **多标签管理**：在一个窗口中通过左侧标签栏管理所有便签，切换便捷，告别桌面杂乱。
*   **智能待办 (Smart Todo)**：
    *   支持复选框（☑/☐）管理待办事项。
    *   **自动排序**：勾选完成的项目会自动移动到便签末尾，未完成事项自动上浮，保持清单整洁。
*   **富文本与样式**：
    *   支持牛皮纸质感背景与行间横线，还原真实书写体验。
    *   支持插入分隔线 (`Ctrl+Enter`)。
    *   支持 `Ctrl+滚轮` 缩放字体大小。
*   **数据安全**：
    *   **自动保存**：输入后短暂防抖自动触发保存。
    *   **原子写入**：采用临时文件替换策略，防止断电或崩溃导致数据损坏。
    *   **数据恢复**：支持从托盘菜单恢复最近删除的便签。
*   **系统集成**：
    *   **系统托盘**：支持后台运行，提供新建、搜索、导出、自启动等快捷入口。
    *   **全局热键**：随时随地通过快捷键唤起便签。
    *   **置顶模式**：支持窗口置顶 (`Ctrl+T`)，方便对照录入。

## ⌨️ 快捷键指南

### 全局热键 (系统任意位置有效)
*   `Ctrl + Alt + N`：新建便签 (并唤起窗口)
*   `Ctrl + Alt + F`：搜索便签

### 应用内快捷键 (当窗口激活时)
*   `Ctrl + N`：新建便签页
*   `Ctrl + D`：删除当前便签页
*   `Ctrl + W`：关闭/隐藏主窗口
*   `Ctrl + T`：切换窗口置顶 (Always on Top)
*   `Ctrl + E`：导出当前便签为文本文件
*   `Ctrl + Enter`：插入水平分隔线
*   `Ctrl + 滚轮`：调整文字大小

## 🚀 构建与运行

### 环境要求
*   Windows 10 / 11
*   .NET SDK 8.0+

### 运行方式
1.  克隆项目到本地。
2.  在项目根目录打开终端。
3.  运行命令：
    ```bash
    dotnet run
    ```
    或者发布为独立应用：
    ```bash
    dotnet publish -c Release -r win-x64 --self-contained
    ```

## 📂 数据存储

*   所有便签数据以 JSON 格式存储于：
    `%APPDATA%\StickyNote\StickyNotes_Data.json`
*   应用会自动管理该文件，建议定期备份该文件以防万一。
